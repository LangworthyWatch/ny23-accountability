This visualization maps the financial network connecting Nick Langworthy to companies that received IDA tax exemptions in New York. Each node is a real entity from public records. Each edge is a documented donation.

**How to use:** Scroll or pinch to zoom. Click and drag to pan. Hover any node for details. The animated version below shows how the network was built year by year.
